# Hexagen Language Support

This extension adds rich Language Server Protocol (LSP) and syntax highlighting for the Hexagen Framework (`.hx` files).

## Features
- **Syntax Highlighting**: Complete highlighting for Hexagen blocks (`slice`, `view`, `api`, `job`) and keywords (`field`, `action`, `title`, etc.).
- **Live Diagnostics**: Red squiggly lines highlight syntax and semantic compilation errors as you type.
- **Auto-Completions**: Real-time suggestions for keywords, types, and user-declared slices.

## Requirements
To run the language server features, make sure the Hexagen CLI (`hf` and `hf_core`) is installed and available in your system `PATH`.
